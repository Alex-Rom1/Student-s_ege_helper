# students_ege_helper

ИИ бот, созданый, чтобы помогать ученикам школ с подготовкой к ЕГЭ

## Getting Started

Для тестового билда нужго запустить файл lib/main.dart
Данный проект использует сервис с API для разных LLM под названием openrouter
Ссылка на сервис: https://openrouter.ai/

Также тут есть приложение example, где будут показаны все темы, шрифты и виджеты
Для его билда надо запустить файл example/lib/main.dart

В папке .release находится готовый APK билд приложения
